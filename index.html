Columns

Like Bootstrap, Foundation also has a pretty killer grid system for making columns and whatnot.

I have divided the screen, 12 columns long, into 8 grid lengths. All subgrids under me will be divided across my 12 columns. That is, if I made a div of column length 6, it'll take up half of my 8 column container, or 4 columns total!

6 columns (4 columns total)

Isn't math fun?

The .container equivalent for Bootstrap in foundation is callout.

They are containers that have easily overwritten styles, to allow for flexility in design.

They have attributes such as primary, secondary, warning, success, and alert for varying visual themes.

Forms

Forms are also pretty easy to make in Foundation. They are made using input tags, with type equal to the type you would like the form to submit. Using placeholder will add text that grays out in the form.

Inputs can have variable sizes using Foundation's grid system. Simply wrap the input in a div with the wanted number of columns.
